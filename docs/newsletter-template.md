# Year of the ZABAL - daily newsletter template

Reusable skeleton for the daily edition. Copy this, fill the brackets, publish to
paragraph.com/@thezao. Keep it scannable - readers should know where to look each day.

## Voice + brand (hard rules)
- BetterCallZaal / Year of the ZABAL voice: direct, warm, builder-energy, short parallel lines.
- No emojis. No em dashes (hyphens only). No crypto/web3/onchain jargon ("builders", "digital creators").
- "100+" for ZAO member count, never a specific number.
- Vary the closing line day to day; the "build is the application" refrain is a recurring anchor, not every day.
- Day number is the day-of-year (e.g. Jun 16 = 167 of 365).

## Pasting to Paragraph
Paragraph's editor accepts pasted markdown and converts it to formatted rich text - `#`
headings, `-` bullets, `**bold**`, and bare URLs auto-link. So write the edition as clean
markdown (this file is), select the block below the rule, and paste. If a paste ever
over-formats, use Cmd+Shift+V to paste as plain text.

## The fixed skeleton (in order)
1. **Greeting + the day's hook** - the one thing that makes today today (Fractal Day, a bounty day, a technical session, a triple-header).
2. **Yesterday in one line + link** - one sentence on the prior session, with its `/recordings/...` link. Never let a session go unlinked.
3. **Quote of the day** - one sharp line pulled from a transcript, attributed, with a click-right-to-the-line deep link (see below).
4. **Today / this week** - today's session first (time, RSVP), then the rest of the week as bullets, then the calendar link.
5. **Insert Coin today** - the one clear action: claim the collectible. `Insert Coin today: claim the ZABAL Gamez collectible - https://collect.zabalgamez.com`
6. **Today's mindful moment** - the daily reflection. Name the title, credit the book (Don't Sweat the Small Stuff, The Four Agreements, etc.), then one builder-facing tie-in. This is a mindful moment from the book, not a "card".
7. **Close + links** - a short sign-off, then the canonical links: recordings, /zabal channel, calendar.

## How to grab a quote-of-the-day link (the deep link)
Every session transcript renders on its recording page with click-to-line deep links:
1. Open the session at `https://zabalgamez.com/recordings/<N>` (or `/recordings/zao/1`, `/recordings/fireside/1`).
2. Find the line you want, hover it, click the `#` that appears - the link is copied to your clipboard.
3. Paste it. It looks like `https://zabalgamez.com/recordings/11#p-49` and opens scrolled to that exact line. If the line has a timestamp it also seeks the video.
