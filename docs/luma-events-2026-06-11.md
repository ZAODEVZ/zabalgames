# Luma events to publish - June 2026

Paste-ready copy for every scheduled session that does not yet have a Luma event.
Luma has no API, so each is created by hand at luma.com/zao. Create them, then paste the
event URLs back and they get wired into `data/workshop-leads.json` (`luma_url`) so the
homepage schedule + top CTA RSVP button light up.

**Title convention (keep consistent):** `ZABAL WORKSHOP w/ <name> from <org>` for
workshops, `ZABAL GameZ FIRESIDE w/ <name> from <org>` for firesides/Spaces (drop "from
<org>" when there is no org). Do not reword titles.

Brand-clean: no emojis, no em dashes (hyphens only), no jargon. Each event is part of the
ZABAL Gamez Season 1 calendar (`luma.com/zao`). Host: ZABAL Gamez. Times are EST.

Standard footer to append to every description:
> ZABAL Gamez is The ZAO's 3-month Build-A-Thon. June workshops, July open build, August
> Finals. Free, open to anyone, any harness. zabalgamez.com
