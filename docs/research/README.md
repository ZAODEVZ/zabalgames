## How to use this library

If you are **a builder onboarding to ZABAL Gamez:** start at [630-player-context-bundle.md](./630-player-context-bundle.md). That is the full ecosystem context you load into your AI tool's project memory.

If you are **a mentor or workshop lead:** start at [701-canonical-state.md](./701-canonical-state.md) for the current shape of the event, then [714-tyler-call.md](./714-tyler-call.md) for the Magnetiq / connector layer.

If you are **a future Season organizer:** start at 701, then read 630 for the original thinking, then 714 + 719 for the meeting decisions that bent the plan.
