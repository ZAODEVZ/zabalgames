# Media kit - ZABAL Gamez

> The press / partner / guest kit: everything someone needs to write about, sponsor, or join
> ZABAL Gamez, in one place. Built per 2026 press-kit practice (a living page, asset-rich, with
> proof and current numbers). Brand rules are hard: no emojis, no em dashes (hyphens only), no
> crypto/web3 jargon in public copy ("digital creators" / "builders"), "100+" for member count.
>
> Keep the numbers current. The recordings count is generated - check recordings/index.json.

## One-liner
ZABAL Gamez is The ZAO's 3-month Build-A-Thon for the Farcaster ecosystem: June workshops, July open build, August Finals. Free, open to anyone, any harness.

## Boilerplate (pick the length you need)
**Short (1 sentence):**
ZABAL Gamez is The ZAO's free, 3-month Build-A-Thon for the Farcaster ecosystem - workshops in June, an open build in July, and Finals in August.

**Medium (3 sentences):**
ZABAL Gamez is The ZAO's 3-month Build-A-Thon for the Farcaster ecosystem. June is a month of recorded workshops from builders and digital creators; July is an open build where anyone ships a project for the ecosystem; August is a curated Finals that crowns the season's best builder. It is free, open to anyone, and harness-agnostic - bring whatever tools you build with.

**Long (paragraph):**
ZABAL Gamez is The ZAO's 3-month Build-A-Thon, built for the Farcaster ecosystem and run by The ZAO, a 100+ member decentralized impact network for the creator economy. It runs across three tracks - artist, builder, and creator - over three phases: June workshops (a recorded library of sessions on the tools and context of the ecosystem), July's open build (anyone ships a project in public), and an August Finals (a cohort curated from July, each paired with a ZAO mentor, ending in a live winner reveal). Every session is livestreamed, transcribed, and kept, so missing it live never means missing it. It is free, open to anyone, and works with any build harness.

- **What:** a 3-month Build-A-Thon (build event), not a video-game contest.
- **Who runs it:** The ZAO - a 100+ member decentralized impact network for the creator economy.
- **Host:** BetterCallZaal (Zaal).
- **When:** June workshops -> July open build -> August Finals. Season 1, 2026.
- **Tracks:** artist (musical / visual), builder (developer / aspiring), creator (media / distribution).
