# ZABAL Gamez Brand Kit - Season 1

> The official brand spec for ZABAL Gamez Season 1. Logo reveal 2026-05-28.

## Logo

The Season 1 logo lives at `/assets/logo.png` (1024x1024 PNG, ~1.2 MB).

Use it for: OG image, Twitter card, Telegram preview, Mini App splash, app icon (when raster suffices), social posts, t-shirts, stream overlays.

Don't use it for: favicon (use `/assets/icon.png` for that - the clean Z mark).

## Visual direction

**Arcade-meets-builder.** The Season 1 logo leans retro arcade cabinet:
- Chunky 3D letter forms (yellow "ZABAL" / red "Z" / cyan "GAMES")
- "INSERT COIN" pixel font caption
- Rainbow pixel-art border (tetris-style multi-color blocks)
- Joystick + button graphics on either side
- CRT scan-line dark navy background

Pair it with: **dark, sleek surfaces** for the actual site UI. The logo carries the retro energy. The chrome stays modern minimalist (existing site dark theme works).

The pairing - retro hero, modern chrome - reads as "playful builder space," not "novelty arcade site."

## Typography

### Site (current, unchanged)

- **Headings:** Syne 700-800 (Google Fonts)
- **Body:** Outfit 300-600
- **Numbers, code, mono:** JetBrains Mono 400-500

### Asset design (optional layer)

For posters, stream overlays, social cards that should LOOK arcade-y:

- **Pixel display font:** Press Start 2P (Google Fonts) or VT323 - use sparingly for "INSERT COIN" treatment, NOT for body copy
- Chunky 3D titles: Google Fonts has `Bungee` or `Russo One` for the ZABAL/GAMES feel
- Body still Outfit for legibility

Do NOT try to put a pixel font in the site body. It reads ironic + hurts legibility. Save it for hero-graphic treatments only.

## Voice

Match these rules across all copy. The voice DOESN'T change because the logo did.

- Direct, warm, builder-energy
- "We learn it together" not "trust us"
- Drop articles when terse helps ("Sign up. Insert coin." NOT "You should sign up by inserting a coin.")
- "Insert coin" is the spirit-totem callback - use it sparingly + intentionally (one moment per surface max)
- No emojis. No em dashes. (Global rule.)

## Approved phrases (rotate sparingly)

- "Insert coin Mon Jun 1"
- "Pick your path. Insert coin."
- "Three doors. All real. Pick yours."
- "Watch. Learn. Build."
- "Free to join. Anyone welcome."
- "Use whatever harness you want."
- "Bring the build, we bring the team + distribution."

## Banned phrases

- "Hackathon" - we are a Build-A-Thon. 3 months, not a weekend. Mentor pairs.
- "Pitch competition" - it's a build event. Working code matters, slides do not.
- "Crypto-only" or "web3 native" in PUBLIC copy - use "digital creators" framing instead
- "Enter the metaverse" - hard pass
- Specific holder counts ("ZAO has 188 members" -> use "100+ members")
- Specific past WaveWarZ stats - they are snapshots, point at the live URL

## Mini App splash sequence

When ZABAL Gamez is opened inside Warpcast:

1. Splash shows `/assets/logo.png` on `#070709` background
2. `assets/miniapp.js` calls `sdk.actions.ready()` to dismiss splash
3. Main page renders

If splash hangs - the `miniapp.js` is the only thing dismissing it. Check it's loaded on the offending page.

## When something doesn't fit

If a logo placement looks weird at small size (Slack channel icon, tiny avatar contexts where the "INSERT COIN" caption is unreadable), use `/assets/icon.png` (the clean Z) instead. Logo is for at-rest hero use; icon is for small-context.
