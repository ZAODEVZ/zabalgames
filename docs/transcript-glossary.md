# Transcript glossary - ZABAL Gamez / ZAO

Canonical spellings and common auto-transcript mishears, used when cleaning workshop,
Bonfire, BCZ YapZ, and bootcamp transcripts/captions. Auto-transcription mangles almost
every proper noun, so apply these as find-and-replace before publishing a transcript or
caption file.

This is a living file - add new names and mishears as each transcript is processed.

## Conventions
- Captions / running prose use title case: **Zaal, Zabal, Zabal Gamez, Diviflyy**.
- The official brand mark in site copy is **ZABAL Gamez** (all caps); the token ticker is
  **$ZABAL**. Use those in repo/site copy; title case is fine in captions.
- The brand uses a **z**: it is **Gamez**, not "Games".
- Brand rules still apply to anything public: no emojis, no em dashes (hyphens only).
- When a name is uncertain, leave it and add it to "Confirm before use" below - do not guess.

## How to use this
1. Before publishing a transcript or caption file, run the global replacements above.
2. Then read for the specific one-off garbles that the table cannot catch by simple
   find-replace (e.g. "is a ball" -> "Zabal").
3. Add any new name/mishear you hit to the right table so the next transcript is faster.
