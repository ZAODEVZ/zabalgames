July - Open Build

Build Days.

Every day in July, one project takes the stage and we build on top of it - live. The room votes on what to make, we build it on stream, then demo it and hand off to tomorrow. Free, recorded, kept. Come build.

Watch live - zabalgamez.com/live

How a Build Day runs

  • 0-10 min. The guest shows their project, then the room votes on what to build on top of it.
  • 10-55 min. We build it live on stream. Suggest, heckle, help when it gets stuck. It doesn't have to finish.
  • 55-60 min. Demo what changed, post the recording, tee up tomorrow's project.

The one rule: everything built that day is on top of that day's project. Register your build any time.
